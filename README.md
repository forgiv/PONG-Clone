# PONG-Clone
A simple PONG clone written in python with pygame api.

REQUIRES:
  Python 2
  Pygame

Player controls:

Player 1: 
W = UP
S = DOWN

Player 2:
UP ARROW = UP
DOWN ARROW = DOWN

Press ESCAPE during game to access settings menu
